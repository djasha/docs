DealDash is a multi-surface platform:

1. React web app (`/client`)
2. Express API (`/server`)
3. Supabase Postgres/Auth
4. WhatsApp MV3 extension (`/whatsapp-extension-react`)
5. LinkShot MV3 extension (`/linkshot-extension`)

## Core capabilities

- Deals, influencers, contacts
- Tasks/reminders/calendar
- Templates and message templates
- Team collaboration and sharing
- Screenshot tracking and short links

## Runtime ports

- Web frontend: `http://localhost:3000`
- API server: `http://localhost:3333`
