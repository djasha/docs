# Contributing

## Scope

This docs set covers DealDash platform, APIs, extensions, operations, and AI agent workflows.

## Pull Request Requirements

- pages are accurate to current behavior
- links pass validation
- docs navigation includes new pages
- risk-sensitive changes include safety notes

## Local Checks

From DealDash source repository:

```bash
pnpm docs:check
```

## Authoring Standards

- concise and task-oriented
- clear command examples
- explicit environment assumptions
- include rollback/recovery notes for operational docs
