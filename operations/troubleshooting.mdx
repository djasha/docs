## API auth failures

- Verify session exists in frontend.
- Confirm bearer token is attached.
- Check `/server/middleware/auth.ts` logs.

## Extension cannot reach backend

- Verify extension host permissions.
- Verify CORS origins in `/server/index.ts`.
- Verify token sync logic and storage keys.
