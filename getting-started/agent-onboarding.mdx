## Roles

- Manager agent: planning, gating, risk control
- Support agent: implementation and validation

## Non-negotiables

1. Run typecheck/tests before merge.
2. Update docs in the same task.
3. No destructive git operations without explicit approval.

Use `/docs/agents/TASK_ROUTING_MATRIX.md` for ownership and validation mapping.
